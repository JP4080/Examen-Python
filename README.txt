Examen Backend - Fundamentos de Python

Este proyecto corresponde al examen de Backend del curso de Fundamentos de Python y Git - SENA CBA Mosquera.

Estructura del proyecto:

ExamenPython/
├── variablesoperaciones/
│     ├── manejo_variables.py → Operaciones básicas con variables, strings, listas y diccionarios.
│     └── tipos_datos.py → Declaración de diferentes tipos de datos y uso de type().
│
├── cicloscondicionales/
│     └── ciclos_condicionales.py → Ejemplos de bucles for, while, condicionales if, elif, else.
│
├── crudnativo/
│     └── crud.py → CRUD básico usando listas (crear, leer, actualizar, eliminar productos).
│
├── investigacionlibreria/
│     └── investigacion_librerias.txt → Investigación sobre la librería tkinter (interfaces gráficas).
│
└── README.txt → Información general del proyecto.

Contenido:

- manejo_variables.py → Operaciones numéricas, comparaciones, lógica, strings, listas y diccionarios.
- tipos_datos.py → Declaración y tipos de datos (int, float, str, bool, list).
- ciclos_condicionales.py → Bucles for, while, condicionales, uso de break, continue.
- crud.py → CRUD básico de productos usando listas.
- investigacion_librerias.txt → Investigación de la librería tkinter, funcionalidad, sintaxis y ejemplo.

Requisitos:
- Tener Python instalado.
- Ejecutar los archivos desde consola o un editor (como Visual Studio Code).
